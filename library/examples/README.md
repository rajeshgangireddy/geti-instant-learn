# Examples

Interactive notebooks and standalone scripts demonstrating Geti Instant Learn capabilities.

## Notebooks

Interactive Jupyter notebooks for learning and experimentation.

| Notebook | Description |
| -------- | ----------- |
| [DINOv3 Zero-Shot Classification](notebooks/dinov3_zero_shot_classification.ipynb) | Zero-shot image classification using DINOv3 with text prompts on FSS-1000 data. |
| [SAM3 Aerial & Maritime](notebooks/sam3_aerial_maritime_example.ipynb) | Zero-shot segmentation on aerial maritime imagery using text and box prompts. |

## Scripts

Standalone Python scripts for benchmarking and comparison.

| Script | Description |
| ------ | ----------- |
| [YOLOE OpenVINO Demo](scripts/yoloe_openvino_demo.py) | Side-by-side PyTorch vs OpenVINO inference with timing comparison. |
| [YOLOE Benchmark](scripts/yoloe_benchmark.py) | Comprehensive benchmark across YOLOE variants and quantization formats. |
| [Compare YOLOE Outputs](scripts/compare_yoloe_outputs.py) | Visual comparison grids across PyTorch and OpenVINO backends (FP32/FP16/INT8/INT4). |
| [Plot YOLOE Benchmark](scripts/plot_yoloe_benchmark.py) | Generate grouped bar charts from benchmark CSV results. |

## Assets

Shared datasets used by the examples above, stored in `assets/`.
